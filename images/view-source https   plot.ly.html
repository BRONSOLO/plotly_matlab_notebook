<section class="feature section__inner">
<header class="text--center push--bottom">
<h3 class="subheading gamma push--bottom">
Import and wrangle.
</h3>
</header>
<div class="feature__img feature__img--right">
<img src="/static/img/splash/data_grid.png">
</div>
<ul class="feature__list">
<li>
Any data format works &mdash; Excel spreadsheets, CSV, TSV, Matlab data files, MS Access, or text. If your format doesn't work, get in touch! You can always reach Plotly on <a href="https://twitter.com/plotlygraphs" class="link--impt" target="_blank">twitter</a> or feedback [at] plot [dot] ly.
</li>
<li>
Bring data in from anywhere: upload from your computer, copy-paste, drag and drop a file, or import from Google Drive and Dropbox.
</li>
<li>
Clean it up fast. Find and replace, convert date-time formats, interpolate, and more.
</li>
</ul>
</section>
<section class="feature section__inner">
<header class="text--center push--bottom">
<h3 class="subheading gamma push--bottom">
Analyze, or simulate.
</h3>
</header>
<div class="feature__img feature__img--left">
<img src="/static/img/splash/best_fit_closeup.png">
</div>
<ul class="feature__list">
<li>
Do the math! Apply any custom function to your data.
</li>
<li>
Derivatives, integrals, fits, and more.
</li>
<li>
Stats galore. Inferential or descriptive basic statistics, chi-squared test, T-test, and ANOVA.
</li>
<li>
Writing a simulation? Code up a python script or use the built-in python sandbox (you can even import any python package, like numpy, scipy, or pandas).
</li>
</ul>
</section>
<section class="feature section__inner">
<header class="text--center push--bottom">
<h3 class="subheading gamma push--bottom">
Graph it.
</h3>
</header>
<div class="feature__img feature__img--right">
<img src="/static/img/splash/histogram.png">
</div>
<ul class="feature__list">
<li>
Scatter, line, bar, area, error bar, heatmap, box, histogram... making graphs is a snap.
</li>
<li>
Interactive plots let you zoom, pan, hover for data values, and more.
</li>
<li>
Export to PNG, SVG, PDF, or EPS.
</li>
<li>
We've got beautiful defaults. No need to stress about colors if that's not your thing.
</li>
<li>
But don't worry, you've got total control. Every parameter of a graph's appearance can be adjusted.
</li>
</ul>
</section>
<section class="feature section__inner">
<header class="text--center push--bottom">
<h3 class="subheading gamma push--bottom">
Share and collaborate.
</h3>
</header>
<div class="feature__img feature__img--left">
<img src="/static/img/splash/email_sharing.png">
</div>
<ul class="feature__list">
<li>
Share and edit your data, graphs and scripts privately with your coworkers, or show the world.
</li>
<li>
Discuss your projects with your team, and get feedback from your teacher or boss.
</li>
<li>
Stay up-to-date with unobtrusive notifications.
</li>
<li>
Version control has your back when things go wrong.
</li>
<li>
The Plotly community is growing! Profiles and search are coming soon.
</li>
</ul>
</section>
<section class="feature section__inner">
<header class="text--center push--bottom">
<h3 class="subheading gamma push--bottom">
Flexibility, built in.
</h3>
</header>
<div class="feature__img feature__img--right">
<img src="/static/img/splash/script_closeup.png">
</div>
<ul class="feature__list">
<li>
Creature of habit when it comes to code? No problem. You can make shareable, beautiful, interactive graphs in your browser from the comfort of your desktop with our <a href="/API" class="link--impt">APIs</a> for <a class="link--impt" href="https://plot.ly/api/python/">Python</a>, <a class="link--impt" href="https://plot.ly/api/matlab/">MATLAB</a>, <a class="link--impt" href="https://plot.ly/api/r/">R</a>, <a class="link--impt" href="https://plot.ly/api/julia/">Julia</a>, <a class="link--impt" href="https://plot.ly/api/rest/">REST</a>, <a class="link--impt" href="https://plot.ly/api/arduino/">Arduino</a>, or <a class="link--impt" href="https://plot.ly/api/perl/">Perl</a>.
</li>
<li>
You can use the Arduino and REST APIs to stream data from hardware devices, too.
</li>
<li>
Secure and privacy-conscious. Built with secure websockets, https, and Amazon S3.
</li>
<li>
Need consistent graph styling? Save a plot theme and apply the look and feel you want, whenever.
</li>
<li>
Plotly is still evolving, so if your needs aren't met, <a onclick="$('#feedbackModal').modal('show');" class="link--impt">drop us a line</a>.
</li>
</ul>
